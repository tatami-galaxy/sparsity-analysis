# Utilities

Shared helpers used across training and analysis.

## Files

- `weight_utils.py` — Save/load weight snapshots, compute parameter diffs, binary masks.
- `math_reward.py` — Answer extraction and verification for MATH (used by GRPO reward function).
- `metrics.py` — Evaluation: accuracy on MATH test set, prior capability benchmarks.
- `logging.py` — Wandb/local logging helpers.
